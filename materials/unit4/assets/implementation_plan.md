# 📊 그래프 단원 도입 자료 3.0 개선 계획

선생님께서 짚어주신 디테일 덕분에 자료가 훨씬 더 완벽해질 것 같습니다! 요청하신 피드백을 반영하기 위한 세부 계획을 정리했습니다.

## Proposed Changes

### 1. 🔙 상시 '이전으로' 버튼 추가
- 각 상황에 진입했을 때, 좌측 상단에 언제든지 메인 메뉴로 돌아갈 수 있는 **[뒤로 가기]** 버튼을 추가하여 조작 편의성을 높이겠습니다.

### 2. 🎬 게임 & 월드컵 "영상 연출" 구현 방안
- **기술적 한계**: 인공지능인 제가 실제 발로란트나 브롤스타즈의 게임 플레이 영상 파일(.mp4)을 실시간으로 창작하여 만들어낼 수는 없습니다.
- **대안 (시네마틱 애니메이션 연출)**: 단순 텍스트 대신, **실제 영상이 재생되는 듯한 엄청난 시각 효과(VFX)**를 코딩으로 구현하겠습니다!
  - **게임 상황**: 화면이 요동치는 흔들림(Shake) 효과, 화면 테두리가 붉어지는 피격 효과, 그리고 화면 중앙에 실제 게임처럼 거대한 **[TEAMMATE ELIMINATED]**, **[YOU DIED]**, **[DEFEAT]** 경고 문구가 화려하게 나타났다가 사라지도록 만듭니다.
  - **월드컵 상황**: 실제 스포츠 중계 화면처럼 거대한 **[선제골 허용]**, **[GOAL!!! 동점골]**, **[GOAL!!! 역전골]** 자막 팝업과 함께 화면이 깜빡이거나 폭죽 효과가 터지도록 연출합니다.

### 3. 📈 x축 주요 순간 레이블 표시
- 단순히 선만 그려지는 것이 아니라, 이벤트가 발생하여 값이 튀어 오르는(Spike) 정확한 **x축 위치 아래에 세로선과 함께 해당 순간의 이름(예: "팀원 데스", "동점골")**이 명확하게 표시되도록 Canvas 렌더링 로직을 대폭 수정합니다.
- 마라탕 그래프에서도 x축에 정확하게 0, 1, 2, 3단계 숫자가 눈금과 함께 표기됩니다.

### 4. 🛠️ 기타 요청 수정 사항
- **월드컵 3번 보기 변경**: "경기 내내 소음이 일정하게 작아질 것이다"로 오답 선택지를 수정합니다.
- **마라탕 배경화면 변경**: 복잡한 사진 배경 대신, 그래프가 눈에 확 띄도록 아주 **깔끔한 하얀색 바탕(White background)**으로 디자인을 변경합니다.

## User Review Required
> [!IMPORTANT]
> **실제 영상 vs 시네마틱 애니메이션 연출**
